# 항공권 예약 가이드

## 항공권 상품 소개

BlockPick에서 제공하는 국내선/국제선 항공권 정보입니다.

### 제공 항공 노선

#### ✈️ 국내선
- 서울(김포/인천) ↔ 제주
- 서울(김포) ↔ 부산
- 서울(김포) ↔ 광주/대구/울산

#### 🌏 국제선 (근거리)
- 일본: 도쿄, 오사카, 후쿠오카, 삿포로, 오키나와
- 중국: 베이징, 상하이, 광저우, 청두
- 동남아: 방콕, 다낭, 세부, 푸켓, 싱가포르
- 기타: 홍콩, 마카오, 타이베이

#### 🌍 국제선 (장거리)
- 미주: LA, 뉴욕, 샌프란시스코, 하와이
- 유럽: 파리, 런던, 로마, 바르셀로나, 프랑크푸르트
- 오세아니아: 시드니, 오클랜드

### 좌석 등급

| 등급 | 영문 | 설명 | 특징 |
|------|------|------|------|
| 일반석 | Economy | 기본 좌석 | 기본 기내식, 짐 23kg |
| 프리미엄 이코노미 | Premium Economy | 넓은 좌석 | 더 넓은 공간, 향상된 식사 |
| 비즈니스석 | Business | 고급 좌석 | 180도 눕는 좌석, 라운지 |
| 일등석 | First | 최고급 | 개인 스위트, VIP 서비스 |

## 예약 프로세스

### 1단계: 당첨 확인
```
항공권 당첨 알림 수신
   ↓
마이페이지 > 당첨 항공권 확인
   ↓
노선 및 좌석 등급 확인
```

### 2단계: 예약 정보 입력 (10일 이내!)

**필수 정보**
```
✅ 탑승 희망 날짜 (왕복: 가는 날/오는 날)
✅ 탑승객 영문 이름 (여권과 100% 일치!)
✅ 생년월일
✅ 성별
✅ 국제선: 여권 번호, 여권 만료일
✅ 연락처 (휴대폰, 이메일)
```

**선택 정보**
```
- 좌석 선호 (창가/통로)
- 특별 기내식 (채식, 유아식 등)
- 휠체어 서비스
- 유아 동반 여부
```

**주의사항** ⚠️
```
✅ 당첨일로부터 10일 이내 입력 필수
✅ 탑승일은 당첨일로부터 14일 ~ 365일 이내
✅ 영문 이름은 여권과 100% 동일해야 함!
   (철자 1자만 틀려도 탑승 불가)
❌ 기한 내 미입력 시 당첨 취소
```

### 3단계: 항공권 발권 (2~3 영업일)

**발권 프로세스**
```
관리자 검수
   ↓
항공사 예약
   ↓
예약 번호 (PNR) 생성
   ↓
전자 항공권 (E-Ticket) 발급
   ↓
이메일 발송
```

**전자 항공권 내용**
```
- 예약 번호 (PNR): ABC123
- 탑승객 이름
- 탑승 날짜 및 시간
- 출발지/도착지
- 좌석 등급
- 수하물 허용량
```

### 4단계: 탑승 수속

#### 온라인 체크인 (출발 24시간 전~)
```
1. 항공사 홈페이지 또는 앱 접속
2. 온라인 체크인 선택
3. 예약 번호(PNR) + 영문 이름 입력
4. 좌석 선택
5. 모바일 탑승권 수령
```

**장점**
```
✅ 공항 대기 시간 단축
✅ 원하는 좌석 우선 선택
✅ 수하물만 맡기면 끝
```

#### 공항 체크인
```
1. 공항 체크인 카운터 방문
2. 여권 + 예약 번호 제시
3. 수하물 위탁
4. 탑승권 수령
5. 보안 검색 → 출국 심사 (국제선)
```

**체크인 마감 시간**
```
국내선: 출발 40분 전
국제선: 출발 60분 전
※ 여유있게 2~3시간 전 도착 권장
```

### 5단계: 탑승
```
게이트 확인 (탑승권에 표기)
   ↓
보안 검색 (기내 수하물)
   ↓
출국 심사 (국제선)
   ↓
게이트 대기
   ↓
탑승 시작 (출발 30~40분 전)
   ↓
좌석 착석
```

## 취소 및 변경 정책

### 취소 정책

#### 운임 등급별 정책

**특가 항공권 (Non-Refundable)**
```
예약 후: 취소/환불 불가
단, 여권 오류 증빙 시 개별 심사
```

**일반 할인 항공권**
```
출발 30일 전: 100% 환불
출발 14일 전: 50% 환불 (50% 수수료)
출발 7일 전: 30% 환불 (70% 수수료)
출발 3일 전~당일: 환불 불가
```

**정상가 항공권 (Flexible)**
```
출발 24시간 전: 100% 환불
출발 24시간 이내: 90% 환불 (10% 수수료)
```

**비즈니스/일등석**
```
출발 3시간 전: 100% 환불
출발 3시간 이내: 90% 환불 (10% 수수료)
```

#### 국내선 vs 국제선

**국내선**
```
취소 가능: 출발 1시간 전까지
수수료: 10,000원 ~ 30,000원
```

**국제선**
```
취소 가능: 출발 24시간 전까지
수수료: $50 ~ $200
Tax/유류할증료: 전액 환불
```

### 변경 정책

#### 날짜 변경

**국내선**
```
변경 가능: 출발 3일 전까지
무료 변경: 1회
2회째부터: 변경 수수료 10,000원
변경 범위: ±7일 이내
```

**국제선**
```
변경 가능: 출발 7일 전까지
무료 변경: 1회
2회째부터: 변경 수수료 $100
변경 범위: ±30일 이내
성수기/비수기 차액 발생 가능
```

#### 노선 변경
```
동일 국가 내: 가능 (차액 정산)
국가 변경: 불가 (취소 후 재예약)
```

#### 이름 변경
```
❌ 원칙: 불가 (IATA 규정)
✅ 오타 수정: 가능 (1~2자 이내, 출발 24시간 전)
   예: HONG GILDONG → HONG KILDONG (O → K)
❌ 전체 변경: 불가 (취소 후 재발권)
```

### 노쇼 정책

**노쇼란?**
```
체크인 마감 시간까지 공항에 나타나지 않는 경우
국내선: 출발 40분 전
국제선: 출발 60분 전
```

**노쇼 페널티**
```
❌ 100% 환불 불가
❌ 왕복 항공권: 가는 편 노쇼 시 오는 편도 자동 취소
```

**노쇼 면제**
```
✅ 천재지변 (태풍, 지진)
✅ 항공사 사유 결항
✅ 긴급 의료 사고 (진단서 제출)
```

## 수하물 규정

### 위탁 수하물 (Checked Baggage)

**일반석**
```
23kg × 1개 (국내선)
23kg × 2개 (국제선 장거리)
```

**비즈니스/일등석**
```
32kg × 2개 (국제선)
```

**초과 수하물**
```
무게 초과: 5,000원 ~ $50/kg
개수 초과: 30,000원 ~ $100/개
```

**금지 물품**
```
❌ 인화성 물질 (라이터 연료, 스프레이)
❌ 예리한 도구 (칼, 가위)
❌ 리튬 배터리 (보조배터리는 기내만)
```

### 기내 수하물 (Carry-on)

**허용 규격**
```
크기: 55cm × 40cm × 20cm 이하
무게: 10kg 이하
개수: 1개 + 개인 소지품 1개
```

**개인 소지품**
```
✅ 노트북 가방
✅ 핸드백
✅ 카메라 가방
```

**액체류 규정**
```
용량: 100ml 이하 용기
총량: 1L 지퍼백 1개
예: 화장품, 향수, 음료
```

## 특별 서비스

### 특별 기내식

**종류**
```
- 채식 (VGML)
- 이슬람식 (MOML)
- 유아식 (BBML)
- 저염식 (LSML)
- 당뇨식 (DBML)
```

**신청**
```
시기: 출발 48시간 전까지
방법: 예약 시 요청 또는 고객센터
```

### 휠체어 서비스

**제공 범위**
```
✅ 공항 체크인 ~ 기내 탑승
✅ 기내 이동 보조
✅ 도착지 게이트 ~ 수하물 수취
```

**신청**
```
시기: 출발 48시간 전까지
증빙: 의사 소견서 (필요 시)
```

### 유아/소아 동반

**유아 (0~23개월)**
```
좌석: 미점유 (보호자 무릎)
요금: 성인 요금의 10%
유모차: 무료 위탁
```

**소아 (2~11세)**
```
좌석: 점유 필수
요금: 성인 요금의 75%
기내식: 어린이 기내식 제공
```

**단독 탑승**
```
만 5~11세: UM 서비스 (동반자 서비스)
만 12세 이상: 단독 탑승 가능
```

## 국제선 특별 안내

### 여권 및 비자

**여권**
```
유효기간: 입국일로부터 6개월 이상
훼손 여부: 훼손 시 출국 불가
사증란: 빈 페이지 2장 이상 권장
```

**비자**
```
무비자 국가: 확인 필수
관광 비자: 사전 발급
전자비자(e-Visa): 온라인 신청
```

### 환승 (Transit)

**환승 시간**
```
최소 환승 시간 (MCT):
- 동일 공항: 1~2시간
- 공항 이동: 3~4시간
권장 시간: 3시간 이상
```

**수하물**
```
직항: 목적지까지 자동 연결
환승: 항공사 정책에 따름
```

### 라운지 이용

**비즈니스/일등석**
```
✅ 무료 라운지 이용
시설: 샤워실, 식음료, 비즈니스 센터
```

**일반석**
```
유료 이용 가능 (우선권 패스 등)
```

## 자주 묻는 질문

### Q1. 영문 이름을 잘못 입력했어요.
**A.** 즉시 고객센터로 연락주세요. 1~2자 오타는 출발 24시간 전까지 무료 수정 가능합니다. 전체 이름 변경은 불가능하며, 취소 후 재발권해야 합니다.

### Q2. 체크인 시간을 놓쳤어요.
**A.** 노쇼 처리되어 환불이 불가능합니다. 항공사에 직접 연락하여 다음 항공편 대기 가능 여부를 문의하세요 (추가 요금 발생 가능).

### Q3. 수하물 무게를 초과했어요.
**A.** 공항에서 초과 수하물 요금을 결제하셔야 합니다. 사전에 무게를 확인하시고, 필요 시 기내 수하물로 분산하세요.

### Q4. 항공편이 지연/결항되었어요.
**A.** 항공사 책임 사유일 경우 항공사에서 직접 보상합니다 (식사, 숙박, 대체 편 제공 등). BlockPick은 중개 역할만 하므로 항공사에 직접 문의하세요.

### Q5. 왕복 항공권인데 가는 편만 취소할 수 있나요?
**A.** 아니요, 왕복 항공권은 일반적으로 분리 취소가 불가능합니다. 가는 편을 노쇼하면 오는 편도 자동 취소됩니다.
