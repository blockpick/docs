# 숙박 상품 예약 가이드

## 숙박 상품 소개

BlockPick에서는 국내외 다양한 숙박 시설을 상품으로 제공합니다.

### 제공 숙박 유형

#### 🏨 5성급 호텔
- 롯데호텔, 신라호텔, 파라다이스호텔
- 포시즌스, 그랜드 하얏트, 웨스틴 조선
- 객실 등급: 디럭스/이그제큐티브/스위트

#### 🏖️ 리조트
- 제주 신라스테이, 파라다이스 부산
- 강원도 알펜시아, 용평리조트
- 경주 힐튼, 여수 디오션

#### 🏡 프리미엄 펜션
- 독채 풀빌라
- 오션뷰 펜션
- 계곡/숲속 힐링 펜션

#### 🌏 해외 숙박
- 동남아 리조트 (발리, 푸켓, 다낭)
- 일본 온천 료칸
- 유럽 부티크 호텔

## 예약 프로세스

### 1단계: 당첨 확인
```
게임 당첨 알림 수신 (이메일/앱)
   ↓
마이페이지 > 당첨 내역 확인
   ↓
상품 정보 확인 (호텔명, 객실 타입, 포함 사항)
```

### 2단계: 예약 정보 입력 (7일 이내 필수!)

**필수 정보**
- 투숙 희망 날짜 (체크인/체크아웃)
- 투숙객 대표자 이름 (실명)
- 연락처 (휴대폰)
- 도착 예정 시간
- 투숙 인원 (성인/어린이)

**선택 정보**
- 침대 타입 (트윈/더블/킹)
- 특별 요청사항 (얼리 체크인, 높은 층 배정 등)
- 기념일 정보 (생일, 기념일 등)

**주의사항** ⚠️
```
✅ 당첨일로부터 7일 이내 입력 필수
✅ 체크인 날짜는 당첨일로부터 3일 ~ 90일 이내
❌ 기한 내 미입력 시 당첨 취소
```

### 3단계: 관리자 검수 (1~3 영업일)

**검수 항목**
- 실명 확인
- 연락처 유효성 검증
- 투숙 가능 여부 확인 (숙소 재고)
- 부정 응모 여부 확인

**검수 통과 시**
```
예약 확정 알림 발송
   ↓
바우처 이메일 발송
   ↓
숙소 예약 번호 생성
```

**검수 반려 시**
```
반려 사유 안내
   ↓
3회까지 재입력 기회
   ↓
3회 초과 시 당첨 취소 (환불 없음)
```

### 4단계: 바우처 수령

**바우처 내용**
```
예약자 이름: 홍길동
호텔명: 제주 신라호텔
예약 번호: BL2024010001
체크인: 2024-02-15 (15:00 이후)
체크아웃: 2024-02-16 (11:00까지)
객실: 디럭스 오션뷰
포함 사항: 조식 2인
```

### 5단계: 호텔 이용

#### 체크인
1. 호텔 프론트 데스크 방문
2. 예약 번호 또는 바우처 제시
3. 신분증 확인
4. 객실 키 수령

#### 투숙
- 호텔 시설 자유 이용
- 룸서비스, 미니바 등은 별도 결제
- 체크아웃 전 추가 요금 정산

#### 체크아웃
- 11:00까지 (레이트 체크아웃 별도 문의)
- 객실 키 반납
- 추가 이용 내역 결제

## 예약 가능 기간

### 체크인 날짜 기준

**일반 상품**
```
당첨일로부터: 3일 ~ 90일 이내
예: 1월 1일 당첨 → 1월 4일 ~ 3월 31일 체크인 가능
```

**성수기 상품** (7~8월, 12~2월)
```
당첨일로부터: 7일 ~ 120일 이내
※ 성수기는 예약이 빠르게 마감될 수 있습니다
```

**특가 상품**
```
날짜 지정: 특정 날짜만 사용 가능
예: 2024년 3월 15일 ~ 3월 16일 (1박 2일)
```

## 객실 타입 및 포함 사항

### 객실 등급

| 등급 | 설명 | 크기 | 뷰 |
|------|------|------|-----|
| 스탠다드 | 기본 객실 | 30㎡ | 시티뷰 |
| 디럭스 | 넓은 객실 | 40㎡ | 시티/마운틴뷰 |
| 이그제큐티브 | 고층 프리미엄 | 50㎡ | 오션/시티뷰 |
| 스위트 | 거실 분리 | 70㎡+ | 프리미엄 뷰 |

### 기본 포함 사항

#### ✅ 모든 객실 포함
- 무료 Wi-Fi
- 욕실 어메니티
- 생수, 차/커피
- 금고, 냉장고
- TV, 헤어드라이어

#### 🍳 조식 포함 여부
상품에 따라 다름 (바우처에 명시)
```
조식 미포함: 객실만 제공
조식 포함: 조식 뷔페 2인
조식 선택: 현장에서 별도 결제 가능
```

#### 🏊 부대시설 이용
대부분 무료 이용 가능
- 피트니스 센터
- 실내 수영장
- 사우나/스파 (일부 유료)
- 비즈니스 센터

## 취소 및 변경 정책

### 취소 정책

#### 체크인 날짜 기준 취소 수수료

**비성수기** (3~6월, 9~11월)
```
15일 전: 100% 환불 (수수료 없음)
14~7일 전: 50% 환불
6~3일 전: 30% 환불
2일 전~당일: 환불 불가
```

**성수기** (7~8월, 12~2월, 공휴일)
```
30일 전: 100% 환불
29~14일 전: 50% 환불
13~7일 전: 30% 환불
6일 전 이후: 환불 불가
```

**특가 상품**
```
예약 즉시: 취소/환불 불가 (Non-Refundable)
단, 천재지변 등 불가항력 시 개별 심사
```

#### 취소 방법
```
마이페이지 > 예약 내역 > 취소 신청
   ↓
취소 사유 입력
   ↓
환불 금액 확인
   ↓
취소 확정
   ↓
환불 처리 (5~7 영업일)
```

### 변경 정책

#### 날짜 변경
```
변경 가능 시점: 체크인 14일 전까지
변경 가능 횟수: 1회 무료
변경 범위: 원래 날짜 ±30일 이내
2회째부터: 변경 수수료 10,000원
```

#### 인원 변경
```
인원 감소: 체크인 7일 전까지 가능 (차액 환불)
인원 증가: 체크인 3일 전까지 가능 (추가 금액 결제)
최대 인원 초과: 불가
```

#### 객실 타입 변경
```
동급 변경: 가능 (숙소 재고에 따름)
상향 변경: 차액 결제 후 가능
하향 변경: 차액 환불 (취소 수수료 적용)
```

### 노쇼 정책

**노쇼란?**
체크인 당일 22:00까지 호텔에 도착하지 않는 경우

**노쇼 페널티**
```
100% 환불 불가
당첨 상품 몰수
```

**노쇼 면제 조건**
```
✅ 사전 연락 (호텔에 직접 전화)
✅ 지연 사유 소명
✅ 새로운 도착 시간 안내
```

## 특별 안내

### 체크인/체크아웃 시간

**기본 시간**
```
체크인: 15:00 이후
체크아웃: 11:00까지
```

**얼리 체크인** (Early Check-in)
```
가능 여부: 호텔 사정에 따름
추가 요금: 50% ~ 100% (반나절/전일)
사전 문의: 체크인 3일 전까지
```

**레이트 체크아웃** (Late Check-out)
```
가능 여부: 호텔 사정에 따름
추가 요금: 시간당 또는 50% ~ 100%
사전 문의: 체크아웃 전날까지
```

### 추가 인원

**규정**
```
기본 인원: 바우처에 명시 (예: 성인 2명)
최대 인원: 객실 타입에 따름

추가 인원 시:
- 성인 추가: 30,000원 ~ 50,000원/1인
- 어린이 추가: 15,000원 ~ 30,000원/1인
```

**어린이 기준**
```
유아 (0~2세): 무료 (침대 미사용 시)
어린이 (3~12세): 추가 요금 (할인)
청소년 (13세 이상): 성인 요금
```

### 반려동물

**펫 프렌들리 호텔**
```
사전 신청: 예약 시 필수 고지
추가 요금: 20,000원 ~ 50,000원/1박
무게 제한: 일반적으로 15kg 이하
제한 구역: 레스토랑, 수영장 등
```

**일반 호텔**
```
반려동물 동반 불가
위반 시 즉시 퇴실 조치
```

### 흡연 정책

```
전 객실 금연
지정 흡연 구역에서만 흡연 가능

객실 내 흡연 적발 시:
- 청소비 100,000원 부과
- 즉시 결제 필수
```

### 주차

**무료 주차**
```
대부분 호텔에서 무료 제공
발레파킹: 일부 유료 (10,000원 ~ 20,000원)
```

**유료 주차**
```
도심지 호텔: 10,000원 ~ 30,000원/1박
바우처 확인: 주차 포함 여부 명시
```

## 해외 숙박 특별 안내

### 추가 준비 사항

**여권**
```
유효기간: 입국일로부터 6개월 이상
훼손되지 않은 여권 필수
```

**비자**
```
국가별 비자 요건 확인
무비자 입국 가능 국가 확인
전자비자(e-Visa) 신청
```

**여행자 보험**
```
권장: 해외 여행자 보험 가입
보장 항목: 질병, 사고, 분실 등
```

### 언어 지원

```
예약 확인서: 영문 제공
체크인: 호텔 자체 언어 (영어/현지어)
긴급 상황: BlockPick 고객센터 (한국어)
```

### 결제

```
추가 요금: 현지 화폐 또는 신용카드
환율: 체크아웃 당일 기준
팁: 국가별 관습에 따름
```

## 자주 묻는 질문 (FAQ)

### Q1. 예약 후 날짜 변경이 가능한가요?
**A.** 네, 체크인 14일 전까지 1회 무료 변경 가능합니다. 2회째부터는 10,000원의 변경 수수료가 부과됩니다.

### Q2. 조식이 포함되어 있는지 어떻게 확인하나요?
**A.** 바우처에 명시되어 있습니다. "조식 2인 포함" 또는 "조식 미포함"으로 표기됩니다.

### Q3. 어린이도 투숙 가능한가요?
**A.** 네, 가능합니다. 단, 객실 최대 인원을 초과할 수 없으며, 3세 이상 어린이는 추가 요금이 발생할 수 있습니다.

### Q4. 예약 후 바우처를 분실했어요.
**A.** 마이페이지에서 다시 다운로드하거나, 고객센터로 연락주시면 재발송해드립니다.

### Q5. 체크인 시간 전에 도착할 것 같아요.
**A.** 호텔에 사전 연락하여 얼리 체크인을 문의하세요. 가능 여부는 호텔 사정에 따라 다르며, 추가 요금이 발생할 수 있습니다.

### Q6. 호텔에서 추가로 결제한 금액은 어떻게 환불받나요?
**A.** 미니바, 룸서비스 등 추가 이용 요금은 BlockPick과 무관하며, 호텔에 직접 문의하셔야 합니다.
