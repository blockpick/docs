# 티켓 상품 예약 가이드

## 티켓 상품 소개

BlockPick에서 제공하는 공연, 스포츠, 전시 티켓 정보입니다.

### 제공 티켓 유형

#### 🎤 공연 티켓
**콘서트**
- K-POP 아이돌 콘서트
- 록/발라드 가수 콘서트
- 클래식/재즈 공연

**뮤지컬**
- 대극장 뮤지컬 (위키드, 오페라의 유령 등)
- 소극장 창작 뮤지컬
- 가족 뮤지컬

**기타 공연**
- 연극, 오페라
- 코미디쇼
- 매직쇼

#### ⚽ 스포츠 티켓
**프로 스포츠**
- 프로야구 (KBO리그)
- 프로축구 (K리그)
- 프로농구 (KBL)
- 프로배구 (V리그)

**국제 경기**
- 월드컵 예선전
- 국가대표 친선경기
- 국제 토너먼트

**기타 스포츠**
- 골프 대회
- 테니스 대회
- e스포츠 대회

#### 🎨 전시/체험 티켓
**전시회**
- 대형 전시회 (팀랩, 미디어아트)
- 미술관/박물관 특별전
- 아트페어

**테마파크**
- 에버랜드, 롯데월드
- 워터파크
- 실내 테마파크

## 예약 프로세스

### 1단계: 당첨 확인
```
당첨 알림 수신
   ↓
마이페이지 > 당첨 티켓 확인
   ↓
공연 정보 확인 (날짜, 장소, 좌석 등급)
```

### 2단계: 예약 정보 입력 (5일 이내!)

**필수 정보**
- 관람 희망 날짜/회차 선택
- 관람자 이름 (실명)
- 연락처
- 티켓 수령 방법 선택

**티켓 수령 방법**
1. 모바일 티켓 (카카오톡/이메일)
2. 실물 티켓 (등기우편)
3. 현장 수령 (공연 당일)

**주의사항** ⚠️
```
✅ 당첨일로부터 5일 이내 입력 필수
✅ 공연일은 당첨일로부터 7일 ~ 180일 이내
❌ 기한 내 미입력 시 당첨 취소
```

### 3단계: 티켓 발권 (1~2 영업일)

**모바일 티켓**
```
공연 1일 전 발송
카카오톡 또는 이메일로 수신
QR 코드 형태
```

**실물 티켓**
```
공연 7일 전 발송
등기우편 (2~3일 소요)
분실 책임은 본인
```

**현장 수령**
```
공연 당일 매표소 방문
예약 번호 + 신분증 지참
공연 시작 1시간 전까지
```

### 4단계: 공연 관람

#### 입장
1. 공연장 방문
2. 티켓 제시 (모바일/실물)
3. 본인 확인 (신분증 - 일부 공연)
4. 좌석 안내

#### 공연 관람
- 공연 시작 시간 엄수
- 공연 중 사진/동영상 촬영 금지
- 휴대폰 전원 OFF 또는 무음

### 스포츠 좌석 등급

**야구장**
```
VIP석: 홈/원정 1루/3루 박스석

지정석: 내야/외야 지정 좌석
자유석: 외야 자유 입장
```

**축구장**
```
VIP석: 메인스탠드 중앙
지정석: 메인스탠드, 사이드스탠드
응원석: 홈/원정 응원 구역
자유석: 일반 지정석
```

## 취소 및 변경 정책

### 취소 정책

#### 공연 날짜 기준 취소 수수료

**콘서트/뮤지컬**
```
30일 전: 100% 환불
29~14일 전: 90% 환불 (10% 수수료)
13~7일 전: 50% 환불 (50% 수수료)
6일 전 이후: 환불 불가
```

**스포츠 경기**
```
14일 전: 100% 환불
13~7일 전: 50% 환불
6일 전 이후: 환불 불가
```

**전시회**
```
관람 기간 시작 7일 전: 100% 환불
관람 기간 시작 후: 환불 불가
```

#### 환불 불가 사유
```
❌ 공연 당일 취소
❌ 티켓 수령 후 분실
❌ 본인 사정으로 미관람
❌ 공연 내용 불만족
```

#### 주최자 귀책 사유
```
✅ 공연 취소: 100% 전액 환불
✅ 공연 연기:
   - 동일 조건 티켓 자동 전환
   - 또는 100% 환불 선택 가능
⚠️ 공연 내용 일부 변경: 환불 불가
```

### 변경 정책

#### 날짜 변경
```
원칙: 불가
예외: 주최 측 공연 일정 변경 시
   → 변경된 날짜로 자동 전환
   → 또는 100% 환불 선택
```

#### 좌석 변경
```
동급 변경: 공연 7일 전까지 1회 가능
상향 변경: 차액 결제 (재고 있을 시)
하향 변경: 불가
```

#### 양도/재판매
```
본인 확인 필요 공연: 양도 불가
본인 확인 불필요 공연:
   → BlockPick 내 양도 시스템 이용
   → 양도 수수료 5%
```

## 입장 및 관람 규정

### 본인 확인

**실명 예매 공연**
```
신분증 지참 필수
예매자 본인만 입장
타인 양도 불가
```

**일반 공연**
```
티켓만 있으면 입장 가능
양도 가능 (BlockPick 시스템 이용)
```

### 연령 제한

```
전체 관람가: 제한 없음
7세 이상: 7세 미만 입장 불가
12세 이상: 12세 미만 입장 불가
15세 이상: 15세 미만 입장 불가
19세 이상: 미성년자 입장 불가
```

**어린이 기준**
```
36개월 미만: 보호자 무릎 관람 (무료)
36개월 이상: 티켓 필요 (유료)
```

### 지각 입장

**일반 규정**
```
공연 시작 후 입장: 주최 측 정책에 따름
대부분: 1부 종료 후 입장 허용
일부: 입장 불가
```

**패널티**
```
❌ 지각으로 인한 미관람: 환불 불가
❌ 시작 후 입장 거부: 환불 불가
```

### 공연장 내 규칙

**금지 사항**
```
❌ 사진/동영상 촬영
❌ 녹음
❌ 음식물 반입
❌ 애완동물 동반
```

**허용 사항**
```
✅ 생수/캔디 (일부 공연장)
✅ 쌍안경
✅ 프로그램 북 구매
```

## 스포츠 관람 특별 안내

### 야구장 규정

**응원 도구**
```
✅ 응원 막대, 메가폰
✅ 응원 유니폼
❌ 악기, 나팔
❌ 위험 물품
```

**음식물 반입**
```
일반적으로 허용
단, 유리병, 캔 등은 제한
구장 내 매점 이용 가능
```

**우천 취소**
```
경기 취소: 100% 환불
경기 중단 (5회 미완료): 100% 환불
경기 중단 (5회 이후): 환불 불가
```

### 축구장 규정

**응원 구역**
```
홈 응원석: 홈팀 응원만
원정 응원석: 원정팀 응원만
중립 구역: 자유 응원
```

**안전 규정**
```
✅ 신분증 지참 (일부 경기)
❌ 불꽃놀이
❌ 위험 물품
```

## 티켓 타입별 안내

### 모바일 티켓

**장점**
```
✅ 분실 위험 없음
✅ 즉시 수령 (공연 1일 전)
✅ 재발급 가능
```

**사용 방법**
```
1. 카카오톡/이메일 수신
2. QR 코드 저장
3. 입장 시 QR 코드 제시
4. 스캔 후 입장
```

**주의사항**
```
⚠️ 스마트폰 배터리 충전 필수
⚠️ QR 코드 캡처 저장 권장
⚠️ 화면 밝기 최대
```

### 실물 티켓

**장점**
```
✅ 수집 가능
✅ 기념품으로 보관
✅ 스마트폰 불필요
```

**주의사항**
```
⚠️ 분실 시 재발급 불가
⚠️ 훼손 시 입장 거부
⚠️ 등기우편 수령 필수
```

### 현장 수령

**수령 장소**
```
공연장 매표소
또는 지정된 현장 발권소
```

**수령 시간**
```
공연 시작 1시간 전부터 가능
공연 시작 직전까지
```

**지참 물품**
```
✅ 예약 번호 (문자 또는 이메일)
✅ 신분증 (실명 예매 시)
```

## 자주 묻는 질문

### Q1. 공연 날짜를 변경할 수 있나요?
**A.** 원칙적으로 불가능합니다. 단, 주최 측에서 공연 일정을 변경하는 경우 변경된 날짜로 자동 전환되거나 100% 환불을 선택할 수 있습니다.

### Q2. 친구에게 티켓을 양도할 수 있나요?
**A.** 본인 확인이 필요한 공연은 양도가 불가능합니다. 일반 공연의 경우 BlockPick 내 티켓 양도 시스템을 통해 양도 가능합니다 (양도 수수료 5%).

### Q3. 지각해서 1부를 못 봤는데 환불받을 수 있나요?
**A.** 아니요, 지각으로 인한 미관람은 환불이 불가능합니다. 공연 시작 시간을 꼭 확인하시고 여유있게 도착해주세요.

### Q4. 모바일 티켓을 분실했어요.
**A.** 마이페이지에서 다시 다운로드하실 수 있습니다. 또는 고객센터로 연락주시면 재발송해드립니다.

### Q5. 야구 경기가 우천 취소되면 어떻게 하나요?
**A.** 경기 취소 시 100% 전액 환불됩니다. 경기가 5회까지 완료되었다면 정식 경기로 인정되어 환불이 불가능합니다.
