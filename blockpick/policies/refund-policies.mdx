# 환불 정책 가이드

## 환불 정책 개요

BlockPick의 모든 상품 및 재화에 대한 환불 규정입니다.

### 환불 가능 항목

```
✅ 게임캐시 & 쇼핑캐시 (게임 참가 이력 없는 경우)
✅ 예약 취소 시 참가비 (수수료 차감)
✅ 주최자 귀책 사유로 인한 취소
✅ 불가항력 사유 (천재지변, 의료 사고 등)
```

### 환불 불가 항목

```
❌ 게임 참가 이력이 있는 경우 (게임캐시 & 쇼핑캐시 모두 환불 불가)
❌ 포인트 (무상 지급 재화)
❌ 노쇼 (No-show)
❌ 본인 사정 미이용 (환불 불가 기간 내)
❌ 부정 사용 적발 계정
```

## 게임캐시 & 쇼핑캐시 환불

### ⚠️ 중요: 게임 참가 이력 기반 환불 정책

```
게임 참가 이력이 환불 가능 여부를 결정합니다.
게임에 한 번이라도 참가하면, 게임캐시와 쇼핑캐시 모두 환불 불가능합니다.
```

### 환불 가능 조건

```
✅ 충전 후 단 한 번도 게임에 참가하지 않은 경우
✅ 충전일로부터 365일 이내
✅ 본인 명의 계좌로 환불
✅ 본인 인증 완료
✅ 게임캐시와 쇼핑캐시 모두 전액 환불 가능
```

**예시: 게임 참가 이력 없음**
```
충전: 100,000원
→ 게임캐시: 100,000원
→ 쇼핑캐시: 100,000원

게임 참가: 0회
쇼핑캐시 사용: 30,000원

환불 가능:
→ 게임캐시: 100,000원 (전액)
→ 쇼핑캐시: 100,000원 (전액)
→ 총 200,000원 환불 (충전 금액의 2배)

⚠️ 게임 참가 이력이 없으면 사용 여부와 무관하게 전액 환불
```

### 환불 불가능 조건

```
❌ 한 번이라도 게임에 참가한 이력이 있는 경우
   → 게임캐시와 쇼핑캐시 모두 환불 불가
   → 사용 여부와 무관

❌ 충전 후 365일 경과
❌ 타인 명의 계좌로 환불 요청
```

**예시: 게임 참가 이력 있음**
```
충전: 100,000원
→ 게임캐시: 100,000원
→ 쇼핑캐시: 100,000원

게임 참가: 1회 (10,000원 사용)

환불 가능:
→ 게임캐시: 0원 (환불 불가)
→ 쇼핑캐시: 0원 (환불 불가)

⚠️ 게임 참가 이력이 있으면 잔액과 무관하게 환불 불가
⚠️ 게임캐시 잔액 90,000원, 쇼핑캐시 100,000원이 남아도 환불 불가
```

### 환불 수수료

```
환불 수수료: 없음
환불 금액: 충전 금액 그대로 환불 (2배 환불)

예시:
충전: 50,000원
환불: 게임캐시 50,000원 + 쇼핑캐시 50,000원 = 100,000원
(수수료 0원)
```

### 환불 절차

#### 1단계: 환불 신청
```
마이페이지 접속
   ↓
결제 내역 > 환불 신청 클릭
   ↓
환불 사유 선택
   ↓
환불 금액 확인
   ↓
환불 계좌 정보 입력
```

#### 2단계: 본인 인증
```
휴대폰 본인 인증 (최초 1회)
   ↓
환불 계좌 명의 = 가입자 실명 확인
   ↓
신분증 사본 업로드 (고액 환불 시)
```

#### 3단계: 환불 처리
```
관리자 검토 (1~2 영업일)
   ↓
환불 승인
   ↓
환불 금액 송금
   ↓
환불 완료 알림 (이메일 + SMS)
```

### 환불 소요 시간

#### 국내 결제 (토스페이먼츠)

**신용카드**
```
당일 결제 취소: 즉시 (1~2분)
승인 취소: 3~7 영업일 (카드사별 상이)

카드사별 환불 일정:
- 삼성카드: 3~5 영업일
- 신한카드: 3~7 영업일
- 현대카드: 5~7 영업일
- KB국민: 3~5 영업일
```

**계좌이체**
```
환불 승인 후: 1~3 영업일
송금 완료 시: SMS 알림
```

**간편결제**
```
토스: 1~3 영업일
카카오페이: 1~3 영업일
네이버페이: 1~3 영업일
```

**휴대폰 소액결제**
```
당월 결제: 즉시 취소
익월 청구 예정: 1~3 영업일 후 계좌 환불
```

#### 해외 결제 (Stripe)

**국제 신용카드**
```
환불 처리: 즉시 (Stripe 처리)
카드사 반영: 5~10 영업일

카드사별 소요 시간:
- Visa: 5~7 영업일
- Mastercard: 5~10 영업일
- American Express: 7~10 영업일
```

**PayPal**
```
환불 처리: 즉시
PayPal 잔액 반영: 24시간 이내
은행 계좌 출금: 3~5 영업일
```

**은행 이체**
```
SEPA (유럽): 3~5 영업일
ACH (미국): 5~7 영업일
```

## 상품 예약 취소 환불

### 숙박 상품 환불

#### 비성수기 (3~6월, 9~11월)

| 취소 시점 (체크인 기준) | 환불 금액 | 취소 수수료 |
|---------------------|----------|-----------|
| 15일 전 | 100% | 0% |
| 14~7일 전 | 50% | 50% |
| 6~3일 전 | 30% | 70% |
| 2일 전~당일 | 0% | 100% |

**예시**
```
참가비: 50,000원
체크인: 2024-03-15
취소일: 2024-03-01 (14일 전)

환불 금액: 25,000원 (50%)
취소 수수료: 25,000원 (50%)
```

#### 성수기 (7~8월, 12~2월, 공휴일)

| 취소 시점 (체크인 기준) | 환불 금액 | 취소 수수료 |
|---------------------|----------|-----------|
| 30일 전 | 100% | 0% |
| 29~14일 전 | 50% | 50% |
| 13~7일 전 | 30% | 70% |
| 6일 전 이후 | 0% | 100% |

**예시**
```
참가비: 100,000원
체크인: 2024-08-01 (성수기)
취소일: 2024-07-05 (27일 전)

환불 금액: 50,000원 (50%)
취소 수수료: 50,000원 (50%)
```

#### 특가 상품

```
❌ 예약 즉시 취소/환불 불가 (Non-Refundable)

예외 (불가항력):
✅ 천재지변
✅ 본인 입원 (진단서 필수)
→ 고객센터 개별 심사 후 100% 환불
```

| 29~14일 전 | 90% | 10% |
| 13~7일 전 | 50% | 50% |

**예시**
```
참가비: 80,000원
공연일: 2024-05-20
취소일: 2024-05-01 (19일 전)

환불 금액: 72,000원 (90%)
취소 수수료: 8,000원 (10%)
```

#### 스포츠 경기

| 취소 시점 (경기일 기준) | 환불 금액 | 취소 수수료 |
|---------------------|----------|-----------|
| 14일 전 | 100% | 0% |
| 13~7일 전 | 50% | 50% |
| 6일 전 이후 | 0% | 100% |

#### 전시회

```
관람 기간 시작 7일 전: 100% 환불

```

### 항공권 상품 환불

#### 특가 항공권 (Non-Refundable)

```
예약 후: 취소/환불 불가

예외 (증빙 필수):
✅ 여권 오류 (출입국 불가)
✅ 비자 거부
✅ 직계 가족 사망
→ 개별 심사 후 환불
```

#### 일반 할인 항공권

| 취소 시점 (출발 기준) | 환불 금액 | 취소 수수료 |
|--------------------|----------|-----------|
| 30일 전 | 100% | 0% |
| 14일 전 | 50% | 50% |
| 7일 전 | 30% | 70% |
| 3일 전~당일 | 0% | 100% |

**Tax/유류할증료**
```
✅ 전액 환불 (항공권 취소 시)
단, 발권 수수료는 공제
```

#### 정상가 항공권 (Flexible)

| 취소 시점 (출발 기준) | 환불 금액 | 취소 수수료 |
|--------------------|----------|-----------|
| 24시간 전 | 100% | 0% |
| 24시간 이내 | 90% | 10% |

#### 비즈니스/일등석

| 취소 시점 (출발 기준) | 환불 금액 | 취소 수수료 |
|--------------------|----------|-----------|
| 3시간 전 | 100% | 0% |
| 3시간 이내 | 90% | 10% |

#### 국내선 vs 국제선

**국내선**
```
취소 가능: 출발 1시간 전까지
취소 수수료: 10,000원 ~ 30,000원
```

**국제선**
```
취소 가능: 출발 24시간 전까지
취소 수수료: $50 ~ $200
```

## 주최자 귀책 사유 환불

### 공연/경기 전체 취소

```
✅ 100% 전액 환불
환불 처리: 자동 환불 (3~5 영업일)
추가 보상: 없음 (원금만 환불)
```

### 공연/경기 일정 연기

**선택 옵션**
```
옵션 1: 변경된 날짜로 티켓 자동 전환
옵션 2: 100% 전액 환불

선택 기한: 공지일로부터 7일 이내
미선택 시: 자동 전환 (옵션 1)
```

### 출연진/선수 변경

```
주연급 변경: 100% 환불 가능 (공지일로부터 3일 이내)
조연급 변경: 환불 불가
```

### 숙박 시설 문제

#### 호텔 폐업/화재

```
✅ 100% 전액 환불
✅ + 동급 이상 호텔 대체 제공 (무료)

대체 호텔 거부 시:
100% 환불만 진행
```

#### 예약 오버부킹

```
호텔 측 실수로 예약 불가 시:

보상 1: 동급 이상 호텔 무료 제공
보상 2: 또는 150% 보상 환불

선택: 고객 결정
```

#### 시설 미달 (사진과 다름)

```
현장 확인 후 부분 환불: 10~50%

절차:
1. 현장 사진 촬영 (증빙)
2. 고객센터 신고
3. 호텔 측 확인
4. 차액 환불
```

### 항공편 지연/결항

#### 항공사 귀책 사유

```
✅ 전액 환불 (BlockPick 처리)
✅ 또는 대체 편 무료 제공 (항공사 처리)

추가 보상 (항공사 직접 제공):
- 식사 쿠폰
- 호텔 숙박 (익일 출발 시)
- 교통편 제공
```

#### 승객 귀책 사유

```
❌ 환불 불가

사유:
- 지각 (체크인 마감 후 도착)
- 서류 미비 (여권, 비자)
- 본인 사정
```

## 불가항력 사유 환불

### 천재지변

**대상 재해**
```
✅ 태풍
✅ 지진
✅ 폭설/한파
✅ 홍수
✅ 화산 폭발
```

**환불 조건**
```
✅ 100% 전액 환불
증빙: 기상청 공식 발표 또는 언론 보도
처리: 증빙 제출 후 1~3 영업일
```

**예시**
```
제주 여행 예정일: 2024-08-15
태풍 '카눈' 제주 상륙: 2024-08-14
기상청 태풍 특보 발령

조치:
→ 100% 전액 환불
→ 또는 날짜 무료 변경
```

### 감염병 (팬데믹)

#### 정부 방역 지침에 따른 휴업

```
공연장/호텔 강제 휴업:
✅ 100% 전액 환불 (자동)
```

#### 본인 격리 대상

```
조건:
✅ 확진 또는 격리 통지서 제출
✅ 격리 기간이 이용 날짜와 겹침

환불:
✅ 100% 전액 환불
```

### 본인 의료 사고

**환불 가능 사유**
```
✅ 입원 (3일 이상)
✅ 수술
✅ 중증 질환 진단
```

**증빙 서류**
```
✅ 진단서 (병원 발급)
✅ 입원 확인서
✅ 수술 확인서

제출 기한: 이용일 3일 전까지
```

**환불 금액**
```
✅ 100% 전액 환불 (수수료 없음)

단, 본인만 인정:
❌ 가족/친구 질병은 불인정
```

**예시**
```
숙박 예정일: 2024-06-10
입원일: 2024-06-07
진단명: 급성 맹장염 수술

조치:
진단서 제출 → 100% 환불
```

### 군 복무 (소집)

**대상**
```
✅ 현역 입영 (훈련소)
✅ 예비군 훈련

```

**증빙 서류**
```
✅ 입영 영장

```

**환불**
```

```

### 직계 가족 사망

**대상**
```

```

```

제출 기한: 이용일 당일까지 가능
```


## 노쇼 (No-Show) 환불 불가

### 노쇼 정의

```
약속 시간까지 나타나지 않는 경우

숙박: 체크인 당일 22:00
티켓: 공연 시작 시간
항공권: 체크인 마감 시간
  - 국내선: 출발 40분 전
  - 국제선: 출발 60분 전
```

### 노쇼 페널티

```
❌ 100% 환불 불가
❌ 당첨 상품 몰수
```

**왕복 항공권 특별 규정**
```
가는 편 노쇼 → 오는 편 자동 취소
   → 전체 환불 불가
```

### 노쇼 면제

**면제 조건**
```
✅ 사전 연락 (호텔/항공사/주최 측)
✅ 지연 사유 소명
✅ 새로운 도착 시간 안내
```

**면제 사유**

## 자주 묻는 질문

### Q1. 게임캐시/쇼핑캐시 환불 시 수수료가 있나요?
**A.** 아니요, 환불 시 수수료는 없습니다. 게임 참가 이력이 없는 경우 게임캐시와 쇼핑캐시를 모두 전액 환불받으실 수 있습니다 (충전 금액의 2배).

### Q1-1. 게임에 한 번 참가하면 환불이 안 되나요?
**A.** ⚠️ 네, 맞습니다. 한 번이라도 게임에 참가한 이력이 있으면 게임캐시와 쇼핑캐시 모두 환불이 불가능합니다. 이는 사용 여부나 잔액과 무관하게 적용되는 정책입니다. 충전 전 신중히 결정해주세요.

### Q1-2. 쇼핑캐시만 사용했는데도 환불이 안 되나요?
**A.** 네, 게임 참가 이력이 있으면 게임캐시와 쇼핑캐시 사용 여부와 무관하게 모두 환불이 불가능합니다. 게임 참가 이력이 환불 가능 여부의 유일한 기준입니다.

### Q2. 카드로 결제했는데 얼마나 걸리나요?
**A.** 카드사별로 다르지만 일반적으로 3~7 영업일이 소요됩니다. 결제 당일 취소하면 승인 취소로 즉시 처리됩니다.

### Q3. 포인트도 환불되나요?
**A.** 아니요, 포인트는 무상 지급 재화이므로 환불이 불가능합니다.

### Q4. 친구 계좌로 환불받을 수 있나요?
**A.** 아니요, 본인 명의 계좌로만 환불 가능합니다. 가족이나 배우자 계좌도 불가능합니다.

### Q5. 노쇼했는데 환불받을 수 있나요?
**A.** 원칙적으로 노쇼는 100% 환불 불가입니다. 단, 사전에 연락하고 정당한 사유(교통사고, 긴급 의료 사고 등)를 증명하면 면제될 수 있습니다.

### Q6. 입원해서 이용 못 했어요. 환불 가능한가요?
**A.** 네, 진단서를 제출하시면 100% 전액 환불 가능합니다. 단, 본인 입원만 인정되며 가족의 질병은 인정되지 않습니다.

### Q7. 환불 신청 후 얼마나 걸리나요?
**A.** 관리자 검토에 1~2 영업일, 승인 후 송금까지 결제 수단별로 1~10 영업일이 소요됩니다.

### Q8. 해외 결제 환불도 가능한가요?
**A.** 네, 가능합니다. Stripe을 통해 환불 처리되며 5~10 영업일이 소요됩니다.

### Q9. 공연이 취소됐어요. 환불은 언제 되나요?
**A.** 주최 측 전체 취소 시 자동으로 3~5 영업일 내에 환불 처리됩니다.

### Q10. 환불 거부 결정에 이의를 제기하고 싶어요.
