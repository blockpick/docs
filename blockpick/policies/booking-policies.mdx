# BlockPick 몰 상품 이용 정책

## 개요

BlockPick 몰에서 쇼핑캐시로 구매한 모든 상품 (숙박, 티켓, 항공권, 명품, 가전)에 적용되는 표준 이용 정책입니다.

## 상품 구매 및 이용 프로세스

### 상품별 이용 가능 기간

| 상품 | 구매 후 예약 기간 | 사용 가능 기간 | 처리 기간 |
|------|---------------|---------------|----------|
| 숙박 | 구매일로부터 7일 이내 | 구매일+3일 ~ +90일 | 1~3 영업일 |
| 티켓 | 구매일로부터 5일 이내 | 구매일+7일 ~ +180일 | 1~2 영업일 |
| 항공권 | 구매일로부터 10일 이내 | 구매일+14일 ~ +365일 | 2~3 영업일 |
| 명품/가전 | 즉시 배송 신청 | 구매일+3일 ~ +30일 | 2~5 영업일 |

### 공통 이용 절차

```
1단계: BlockPick 몰에서 상품 구매 (쇼핑캐시 사용)
   ↓
2단계: 상품 정보 입력 (예약일, 배송지 등)
   ↓
3단계: 관리자 확인
   ↓
4단계: 예약 확정 또는 배송 시작
   ↓
5단계: 상품 이용 또는 수령
```

## 예약 정보 입력

### 필수 정보

#### 숙박 상품
```
✅ 체크인/체크아웃 날짜
✅ 투숙객 대표자 실명
✅ 연락처 (휴대폰, 이메일)
✅ 도착 예정 시간
✅ 투숙 인원 (성인/어린이)
```

#### 티켓 상품
```
✅ 관람 희망 날짜/회차
✅ 관람자 실명
✅ 연락처
✅ 티켓 수령 방법 (모바일/실물/현장)
```

#### 항공권 상품
```
✅ 탑승 희망 날짜 (왕복: 가는 날/오는 날)
✅ 탑승객 영문 이름 (여권과 100% 일치!)
✅ 생년월일, 성별
✅ 국제선: 여권 번호, 여권 만료일
✅ 연락처 (휴대폰, 이메일)
```

### 선택 정보

#### 숙박 상품
```
- 침대 타입 (트윈/더블/킹)
- 특별 요청 (얼리 체크인, 높은 층 등)
- 기념일 정보
```

#### 티켓 상품
```
- 좌석 선호 (특별 요청사항)
```

#### 항공권 상품
```
- 좌석 선호 (창가/통로)
- 특별 기내식
- 휠체어 서비스
- 유아 동반 여부
```

## 구매 후 정보 입력 정책

### 기한 내 미입력 시

```
❌ 구매 자동 취소
❌ 쇼핑캐시 환불 (취소 수수료 적용 가능)
```

**알림 발송**
```
D-3: 이메일 + 앱 푸시
D-1: 이메일 + 앱 푸시 + SMS
D-day: 최종 알림 SMS
```

### 기한 연장

```
원칙: 연장 불가

예외 (불가항력):
✅ 천재지변 (지진, 태풍)
✅ 본인 입원 (진단서 제출)
→ 고객센터 개별 심사
```

## 관리자 검수 정책

### 검수 항목

**공통 검수**
```
1. 실명 확인
2. 연락처 유효성 확인
3. 부정 응모 여부 확인
4. 중복 당첨 여부 확인
```

**상품별 검수**
```
숙박: 투숙 가능 여부 (재고 확인)
티켓: 공연 일정 확인, 좌석 재고 확인
항공권: 여권 정보 정확성, 노선 운항 여부
```

### 검수 결과

#### 검수 통과
```
예약 확정 알림 발송
   ↓
바우처 이메일 발송
   ↓
예약 번호 생성
```

#### 검수 반려
```
반려 사유 안내 (이메일 + SMS)
   ↓
재입력 기회 제공 (최대 3회)
   ↓
3회 초과 시 당첨 취소 (환불 없음)
```

**반려 사유 예시**
```
- 실명 불일치
- 연락처 오류
- 여권 정보 오류 (항공권)
- 예약 불가 날짜 선택
- 중복 예약 시도
```

## 예약 변경 정책

### 날짜 변경

#### 숙박
```
변경 가능 시점: 체크인 14일 전까지
변경 가능 횟수: 1회 무료, 2회째부터 10,000원
변경 범위: 원래 날짜 ±30일 이내
```

#### 티켓
```
원칙: 불가
예외: 주최 측 공연 일정 변경 시
   → 변경된 날짜로 자동 전환
   → 또는 100% 환불 선택
```

#### 항공권
```
국내선:
  변경 가능: 출발 3일 전까지
  무료 변경: 1회
  변경 범위: ±7일 이내

국제선:
  변경 가능: 출발 7일 전까지
  무료 변경: 1회
  변경 범위: ±30일 이내
  성수기 차액 발생 가능
```

### 인원 변경

#### 숙박
```
인원 감소: 체크인 7일 전까지 (차액 환불)
인원 증가: 체크인 3일 전까지 (추가 결제)
최대 인원 초과: 불가
```

#### 티켓
```
원칙: 불가 (1인 1티켓 원칙)
```

#### 항공권
```
원칙: 불가 (탑승객 변경 불가)
```

### 상품 등급 변경

#### 숙박 (객실 타입)
```
동급 변경: 가능 (재고에 따름)
상향 변경: 차액 결제 후 가능
하향 변경: 차액 환불 (취소 수수료 적용)
```

#### 티켓 (좌석 등급)
```
동급 변경: 공연 7일 전까지 1회 가능
상향 변경: 차액 결제 (재고 있을 시)
하향 변경: 불가
```

#### 항공권 (좌석 등급)
```
원칙: 불가
예외: 항공사 정책에 따라 개별 문의
```

## 주최자/항공사 귀책 사유 처리

### 공연/경기 취소/연기

#### 전체 취소
```
✅ 100% 전액 환불
처리: 자동 환불 (3~5 영업일)
```

#### 일정 연기
```
선택 1: 변경된 날짜로 티켓 자동 전환
선택 2: 100% 환불

선택 기한: 공지일로부터 7일 이내
```

#### 출연진 변경
```
주연급 변경: 환불 가능 (공지일로부터 3일 이내)
조연급 변경: 환불 불가
```

### 숙박 시설 귀책 사유

```
호텔 폐업/화재:
  → 100% 환불 + 동급 호텔 대체 제공

예약 오버부킹:
  → 동급 이상 호텔 무료 제공
  → 또는 150% 보상

시설 미달:
  → 현장 확인 후 부분 환불 (10~50%)
```

### 항공편 지연/결항

```
항공사 귀책 사유:
  ✅ 전액 환불
  ✅ 또는 대체 편 무료 제공
  ✅ 항공사에서 직접 보상
     (식사, 숙박, 교통편)

승객 귀책 사유:
  ❌ 환불 불가
```

**BlockPick 역할**
```
중개자 역할만 수행
항공사 직접 보상 안내
필요 시 항공사 연락 지원
```

## 자주 묻는 질문

### Q1. 예약 정보 입력을 깜빡했어요. 기한 연장이 가능한가요?
**A.** 아니요, 예약 입력 기한은 연장이 불가능합니다. 구매일로부터 정해진 기한 내에 반드시 입력해주셔야 하며, 미입력 시 자동 취소됩니다.

### Q2. 예약한 날짜를 변경할 수 있나요?
**A.** 상품별로 변경 정책이 다릅니다. 숙박은 체크인 14일 전까지 1회 무료 변경 가능하며, 티켓은 원칙적으로 불가능합니다. 항공권은 출발 3~7일 전까지 변경 가능합니다.

### Q3. 친구에게 양도할 수 있나요?
**A.** 숙박은 10,000원의 수수료로 양도 가능하고, 티켓은 본인 확인이 필요 없는 공연에 한해 5% 수수료로 양도 가능합니다. 항공권은 양도가 불가능합니다.

### Q4. 취소 수수료는 언제부터 발생하나요?
**A.** 상품과 시기에 따라 다릅니다. 숙박은 체크인 15일 전(비성수기) 또는 30일 전(성수기)까지 무료 취소 가능하며, 티켓과 항공권도 각각 다른 기준이 적용됩니다.

### Q5. 노쇼하면 어떻게 되나요?
**A.** 100% 환불이 불가능하며 당첨 상품이 몰수됩니다. 단, 사전에 연락하고 지연 사유를 소명하면 면제될 수 있습니다.

### Q6. 천재지변으로 이용할 수 없게 되었어요.
**A.** 태풍, 지진 등 천재지변 시 100% 환불 또는 날짜 무료 변경이 가능합니다. 기상청 공식 발표나 뉴스 기사를 증빙자료로 제출해주세요.

### Q7. 미성년자도 예약할 수 있나요?
**A.** 만 14세 미만은 법정대리인 동의가 필수이며, 만 14~19세는 본인 예약이 가능하나 일부 상품(19세 이상 공연, 단독 숙박 등)은 제한됩니다.

### Q8. 개인정보는 어떻게 관리되나요?
**A.** 예약에 필요한 최소한의 정보만 수집하며, 숙박 시설/주최사/항공사에 제공됩니다. 이용 완료 후 5년간 보관 후 파기됩니다.
