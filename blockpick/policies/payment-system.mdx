# 결제 시스템 가이드

## 결제 시스템 소개

BlockPick은 국내외 사용자를 위한 안전하고 편리한 결제 시스템을 제공합니다.

### 결제 서비스 제공사

#### 🇰🇷 국내 결제: 토스페이먼츠
- 한국 1위 간편결제 플랫폼
- 다양한 결제 수단 지원
- 빠르고 안전한 결제 경험
- 즉시 환불 지원

#### 🌏 해외 결제: Stripe
- 글로벌 1위 결제 플랫폼
- 135개 이상 통화 지원
- 다국어 결제 환경
- 국제 카드 및 다양한 결제 수단

## 재화 시스템

BlockPick은 **2가지 캐시 + 1가지 포인트** 시스템을 운영합니다.

### 충전 방식
```
실제 돈으로 충전하면 2가지 캐시를 동시에 지급합니다.

예시: 10,000원 충전 시
→ 게임캐시 10,000원 지급
→ 쇼핑캐시 10,000원 지급
(총 20,000원 상당의 재화 획득)

최소 충전 금액: 1,000원
최대 충전 금액: 1,000,000원/회
```

### 💰 게임캐시 (Game Cash)

**획득 방법**
```
실제 돈을 충전하여 획득 (충전 금액만큼 지급)
```

**사용 용도**
- 게임 참여 전용 (참가비 결제)
- 포인트와 혼합 사용 불가

**환불 정책**
```
⚠️ 중요: 게임 참가 이력이 환불 가능 여부를 결정합니다.

✅ 환불 가능한 경우:
   - 충전 후 단 한 번도 게임에 참가하지 않은 경우
   - 전액 환불 가능
   - 환불 수수료: 없음

❌ 환불 불가능한 경우:
   - 한 번이라도 게임에 참가한 이력이 있는 경우
   - 게임캐시와 쇼핑캐시 모두 환불 불가

⚠️ 환불 제한: 충전 후 365일 이내
```

**만료 기간**
```
충전일로부터 5년
만료 30일 전 알림 발송
```

### 🛍️ 쇼핑캐시 (Shopping Cash)

**획득 방법**
```
실제 돈을 충전하여 획득 (충전 금액만큼 지급)
```

**사용 용도**
- BlockPick 몰에서 여행상품 구매 전용
- 포인트와 혼합 사용 가능

**환불 정책**
```
⚠️ 중요: 게임 참가 이력이 환불 가능 여부를 결정합니다.

✅ 환불 가능한 경우:
   - 충전 후 단 한 번도 게임에 참가하지 않은 경우
   - 전액 환불 가능
   - 환불 수수료: 없음

❌ 환불 불가능한 경우:
   - 한 번이라도 게임에 참가한 이력이 있는 경우
   - 게임캐시와 쇼핑캐시 모두 환불 불가

⚠️ 환불 제한: 충전 후 365일 이내
```

**만료 기간**
```
충전일로부터 5년
만료 30일 전 알림 발송
```

### 🎁 포인트 (Point)

**획득 방법**
```
1. 게임 낙첨 시 자동 적립
   → 참가비의 10%

2. 이벤트 참여
   → 출석 체크 (7일: 1,000P, 30일: 5,000P)
   → 친구 추천 (가입: 3,000P, 첫 게임: 2,000P)

3. 상품 후기 작성
   → 텍스트 후기: 500P
   → 사진 후기: 1,000P

4. 신규 가입 혜택
   → 가입 즉시 5,000P 지급
```

**사용 용도**
```
✅ BlockPick 몰: 사용 가능 (현금성 할인)
❌ 게임 참여: 사용 불가
```

**환불 정책**
```
❌ 환불 불가 (무상 지급 재화)
```

**만료 기간**
```
적립일로부터 1년
만료 30일 전 알림 발송
```

### 재화 사용 규칙

#### 게임 참여
```
사용 가능: 게임캐시만 가능
사용 불가: 쇼핑캐시, 포인트
```

#### BlockPick 몰 (여행상품 구매)
```
사용 가능: 쇼핑캐시, 포인트
사용 불가: 게임캐시

혼합 사용 가능:
✅ 쇼핑캐시 + 포인트

사용 우선순위:
1순위: 포인트 (먼저 소진)
2순위: 쇼핑캐시 (포인트 부족 시)
```

**예시: BlockPick 몰 구매**
```
30,000원 여행상품 구매 시

보유 재화:
- 포인트: 5,000원
- 쇼핑캐시: 50,000원
- 게임캐시: 10,000원

자동 결제:
- 포인트 5,000원 사용
- 쇼핑캐시 25,000원 사용
- 게임캐시: 사용 불가
→ 총 30,000원 결제 완료
```

## 국내 결제: 토스페이먼츠

### 지원 결제 수단

#### 💳 신용카드/체크카드
```
지원 카드사:
- 삼성, 신한, 현대, KB국민, 우리, 하나
- BC, 롯데, NH농협, 씨티

특징:
✅ 즉시 결제
✅ 할부 결제 가능 (2~12개월)
✅ 카드사 포인트 사용 가능
```

**할부 안내**
```
5만원 이상: 2~3개월 무이자
10만원 이상: 2~6개월 무이자 (카드사별 상이)
20만원 이상: 2~12개월 무이자 (이벤트 기간)
```

#### 🏦 계좌이체
```
실시간 계좌이체
모든 은행 지원

특징:
✅ 수수료 무료
✅ 즉시 충전
✅ 이체 한도: 은행별 상이
```

#### 📱 간편결제
```
- 토스
- 카카오페이
- 네이버페이
- 페이코
- SSG페이

특징:
✅ 원클릭 결제
✅ 생체 인증 (지문, 안면)
✅ 포인트/쿠폰 자동 적용
```

#### 📲 휴대폰 소액결제
```
월 한도:
- 만 19세 미만: 월 최대 30만원
- 만 19세 이상: 월 최대 50만원

특징:
✅ 카드 없이 결제
❌ 통신사 수수료 발생 (약 3~5%)
```

### 결제 프로세스

#### 1단계: 충전 금액 선택
```
빠른 충전:
□ 5,000원
□ 10,000원
□ 30,000원
□ 50,000원
□ 100,000원

직접 입력:
[______] 원 (1,000원 ~ 1,000,000원)
```

#### 2단계: 결제 수단 선택
```
토스페이먼츠 결제창 호출
   ↓
결제 수단 선택
- 신용카드
- 계좌이체
- 간편결제
- 휴대폰
   ↓
결제 정보 입력
```

#### 3단계: 결제 승인
```
결제 승인 처리
   ↓
결제 완료 알림
   ↓
게임캐시 + 쇼핑캐시 즉시 충전
   (충전 금액의 2배 지급)
   ↓
충전 내역 이메일 발송
```

**충전 예시**
```
10,000원 결제 시:
→ 게임캐시 10,000원 충전
→ 쇼핑캐시 10,000원 충전
(총 20,000원 상당의 재화 획득)
```

**소요 시간**
```
카드/간편결제: 즉시 (1~2초)
계좌이체: 즉시 (5~10초)
휴대폰: 1~2분
```

### 결제 보안

**토스페이먼츠 보안 기술**
```
✅ PCI-DSS Level 1 인증
✅ 3D Secure 2.0 (3DS 2.0)
✅ 토큰화 (Tokenization)
✅ SSL/TLS 암호화 통신
```

**추가 보안**
```
- OTP 인증 (선택)
- 생체 인증 (지문, 안면)
- 이메일 알림 (결제 시마다)
- 결제 비밀번호
```

```

주요 통화:
- USD (미국 달러)
- EUR (유로)
- JPY (일본 엔)
- CNY (중국 위안)
- GBP (영국 파운드)
```

#### 2단계: 충전 금액 입력
```
최소 충전: $10 (또는 동등 금액)
최대 충전: $10,000 (또는 동등 금액)

환율:
실시간 환율 적용
Stripe 환율 수수료: 약 1~2%
```

#### 3단계: 결제 정보 입력
```
Stripe Checkout 호출
   ↓
결제 수단 선택
   ↓
카드 정보 입력 또는 간편결제 선택
   ↓
결제 승인
   ↓
게임캐시 + 쇼핑캐시 충전 (환율 적용)
```

**예시**
```
충전 금액: $100
환율: 1 USD = 1,300 KRW

충전되는 재화:
→ 게임캐시: 130,000원
→ 쇼핑캐시: 130,000원
(총 260,000원 상당의 재화 획득)

(환율은 결제 시점 기준)
```

### 환율 정책

**환율 적용**
```
기준: Stripe 실시간 환율
갱신: 실시간 (결제 시점 기준)
수수료: Stripe 환율 마진 약 1~2% 포함
```

**환율 표시**
```
충전 전 예상 환율 안내:
예: $100 ≈ 130,000원
    (환율: 1 USD = 1,300 KRW)

실제 충전 금액은 결제 승인 시점 환율로 확정
```

### 결제 보안

**Stripe 보안 기술**
```
✅ PCI-DSS Level 1 인증
✅ SCA (Strong Customer Authentication)
✅ 3D Secure 2.0

✅ 256-bit SSL/TLS 암호화
```

**추가 보안**

## 환불 정책

### 게임캐시 & 쇼핑캐시 환불

#### ⚠️ 중요: 게임 참가 이력 기반 환불 정책

```
게임 참가 이력이 환불 가능 여부를 결정합니다.
게임에 한 번이라도 참가하면, 게임캐시와 쇼핑캐시 모두 환불 불가능합니다.
```

#### 환불 가능 조건
```
✅ 충전 후 단 한 번도 게임에 참가하지 않은 경우
✅ 충전 후 365일 이내
✅ 본인 명의 계좌로만 환불
✅ 게임캐시와 쇼핑캐시 모두 전액 환불 가능
```

#### 환불 불가능 조건
```
❌ 한 번이라도 게임에 참가한 이력이 있는 경우
   → 게임캐시와 쇼핑캐시 모두 환불 불가
   → 사용 여부와 무관

❌ 충전 후 365일 경과
❌ 타인 명의 계좌로 환불 요청
```

#### 예시
```
케이스 1: 게임 참가 이력 없음
충전: 100,000원
→ 게임캐시 100,000원
→ 쇼핑캐시 100,000원
게임 참가: 0회
환불 가능: 게임캐시 100,000원 + 쇼핑캐시 100,000원 (전액)

케이스 2: 게임 참가 이력 있음
충전: 100,000원
→ 게임캐시 100,000원
→ 쇼핑캐시 100,000원
게임 참가: 1회 (10,000원 사용)
환불 가능: 0원 (게임캐시, 쇼핑캐시 모두 환불 불가)

⚠️ 게임 참가 이력이 있으면 잔액과 무관하게 환불 불가
```

#### 환불 수수료
```
환불 수수료: 없음
환불 금액: 충전 금액 그대로 환불 (2배 환불)
```

### 환불 절차

#### 1단계: 환불 신청
```
마이페이지 > 결제 내역 > 환불 신청
   ↓
환불 사유 선택
환불 금액 확인
환불 계좌 입력
```

#### 2단계: 본인 인증
```
신분증 인증 (1회만)
환불 계좌 명의 = 가입자 실명
```

#### 3단계: 환불 처리
```
신청 검토 (1~2 영업일)
   ↓
환불 승인
   ↓
환불 금액 송금
```

**환불 소요 시간**
```
카드 결제:
- 승인 취소: 즉시 (결제 당일)
- 일반 환불: 3~7 영업일 (카드사별 상이)

계좌이체:
- 1~3 영업일

간편결제:
- 1~3 영업일

해외 결제 (Stripe):
- 5~10 영업일 (카드사별 상이)
```

### 환불 불가 사유

```
❌ 게임 참가 이력이 있는 경우 (가장 중요!)

❌ 충전 후 365일 경과
❌ 부정 사용으로 적발된 계정
❌ 타인에게 양도한 재화
```

### 포인트 환불

```
❌ 포인트는 환불 불가
이유: 무상 지급 재화
```

```
- 카드 정보 오류
```

```
- 서버 지연
```

```
   → 승인 번호 제공
```

**원인**


**해결 방법**

## 자주 묻는 질문

### Q1. 최소 충전 금액은 얼마인가요?
**A.** 국내 결제는 1,000원, 해외 결제는 $10 (또는 동등 금액)부터 충전 가능합니다.

### Q2. 캐시 환불이 가능한가요?
**A.** ⚠️ 중요: 게임 참가 이력이 환불 가능 여부를 결정합니다. 한 번이라도 게임에 참가한 이력이 있으면 게임캐시와 쇼핑캐시 모두 환불이 불가능합니다. 게임 참가 이력이 없는 경우에만 충전 후 365일 이내 전액 환불 가능하며, 환불 수수료는 없습니다.

### Q3. 포인트도 환불되나요?
**A.** 아니요, 포인트는 무상 지급 재화이므로 환불이 불가능합니다.

### Q4. 해외 결제 시 환율이 어떻게 적용되나요?
**A.** Stripe 실시간 환율이 적용되며, 결제 승인 시점의 환율로 확정됩니다. 환율 수수료 약 1~2%가 포함됩니다.

### Q5. 카드 할부가 가능한가요?
**A.** 국내 카드는 5만원 이상 충전 시 할부가 가능합니다 (2~12개월). 해외 카드는 할부가 지원되지 않습니다.

### Q6. 결제는 됐는데 재화가 충전되지 않았어요.
**A.** 5분 후 다시 확인해주세요. 일시적인 지연일 수 있습니다. 계속 미반영 시 고객센터로 승인 번호와 함께 문의해주시면 즉시 처리해드립니다. 정상 충전 시 게임캐시와 쇼핑캐시가 각각 충전 금액만큼 지급됩니다.

### Q7. 현금영수증 발급이 가능한가요?
**A.** 네, 마이페이지에서 주민등록번호 또는 사업자등록번호를 등록하면 자동 발급됩니다. 충전 후 7일 이내 수동 신청도 가능합니다.

### Q8. 해외에서도 사용할 수 있나요?
**A.** 네, Stripe을 통해 전 세계 어디서나 결제 가능합니다. 135개 이상의 통화를 지원하며, 국제 카드 및 PayPal, Apple Pay 등을 사용할 수 있습니다.
