# BlockPick 서비스 소개

## BlockPick이란?

BlockPick은 **블록체인 기술로 투명성을 보장하는 경품 게임 플랫폼**입니다. 사용자가 충전한 캐시를 사용해 격자판 형태의 블록 중 원하는 블록을 선택하고, 정해진 시간에 게임 종료 시 **아무도 고르지 않은 블록을 자신만 선택했을 경우** 승리하는 블록체인 기반 서비스입니다.

### 핵심 차별점

#### 1. 블록체인 기반 투명성 🔐
- 모든 게임 결과가 **Polygon 블록체인**에 영구 기록
- 조작 불가능한 공정한 게임
- 누구나 [PolygonScan](https://polygonscan.com)에서 직접 검증 가능

#### 2. Commit-Reveal 암호학 기법 ⚖️
```
1단계 (Commit): 선택을 암호화하여 제출
   → 다른 사람이 무엇을 선택했는지 알 수 없음

2단계 (Reveal): 모두의 선택을 동시에 공개
   → 따라하기 불가능, 선택 변경 불가능
```

#### 3. 다양한 프리미엄 상품 🎁
- **숙박**: 제주 신라호텔, 파라다이스 부산, 서울 포시즌스 등
- **티켓**: BTS 콘서트, 뮤지컬 '위키드', 야구 VIP석 등
- **항공권**: 제주도 왕복, 도쿄 직항, 파리 비즈니스석 등
- **명품**: 루이비통, 샤넬, 에르메스 가방 / 롤렉스, 오메가 시계 등
- **가전**: 아이폰, 맥북, 갤럭시 / 다이슨 청소기, 공기청정기 등

## 서비스 구조

### 게임 상품 카테고리

#### 🏨 숙박 상품
**국내 숙박**
- 5성급 호텔 (롯데, 신라, 파라다이스 등)
- 프리미엄 리조트 (제주, 강원도, 부산)
- 독채 펜션 및 풀빌라

**해외 숙박**
- 동남아 리조트 (발리, 푸켓, 세부)
- 일본 온천 료칸
- 유럽 부티크 호텔

**상품 구성**
```
- 1박 2일 ~ 3박 4일
- 조식 포함 여부
- 객실 등급 (스탠다드/디럭스/스위트)
- 최대 투숙 인원
```

#### 🎫 티켓 상품
**공연**
- 콘서트 (K-POP, 클래식, 재즈)

**스포츠**

- 골프 대회, 테니스 대회

**전시/체험**
- 대형 전시회 (팀랩, 고흐전 등)
- 테마파크 (에버랜드, 롯데월드)
- 워터파크, 스키장

**좌석 등급**

## BlockPick 몰

### 쇼핑캐시로 상품 구매

게임과 별개로 **BlockPick 몰**에서 쇼핑캐시를 사용해 상품을 직접 구매할 수 있습니다.

```
충전 시: 게임캐시 + 쇼핑캐시 동시 지급
게임캐시 → 게임 참여 전용
쇼핑캐시 → BlockPick 몰 쇼핑 전용
```

**구매 가능 상품**
- 숙박: 호텔, 리조트, 펜션
- 티켓: 콘서트, 뮤지컬, 스포츠, 전시회
- 항공권: 국내선, 국제선
- 명품: 가방, 시계, 액세서리
- 가전: 스마트폰, 노트북, 생활가전

**결제 방식**
```
쇼핑캐시 단독 사용 또는
쇼핑캐시 + 현금 혼합 결제 가능
```

## 게임 방식

BlockPick은 크게 **3가지 게임 모드**를 제공합니다.

### 1. Daily 모드 🎯

**하나의 경품만을 대상으로 진행하는 게임**

#### 게임 구성
```
격자 크기: 3×3 ~ 40×40 (경품 가치에 따라 다름)
픽당 가격: 100원 ~ 100,000원
경품: 단일 경품 (호텔, 티켓, 항공권 등)
당첨자: 1명
```

#### 당첨 규칙
1. **아무도 고르지 않은 블록**을 자신만 선택한 사람 승리
2. 해당 블록이 여러 개일 경우, **가장 빨리 픽한 사람** 당첨
3. 동시 픽 시 **블록체인 타임스탬프**로 결정

#### 예시
```
제주 신라호텔 1박 (10만원 상당)
격자: 10×10 (100블록)
픽당 가격: 1,000원
응모 시간: 24시간

블록별 선택 현황 (응모 마감 후):
블록 7번: 0명 선택 ← 아무도 선택 안 함!
블록 23번: 1명 선택 (참가자 A) ← 혼자 선택!
블록 45번: 3명 선택
블록 67번: 2명 선택
...

결과: 블록 23번을 선택한 참가자 A 당첨! 🎉
→ 제주 신라호텔 1박권 획득
```

#### 게임 유의사항
```
✅ 만 14세 이상만 참여 가능
✅ 경품은 대한민국 내에서만 배송
✅ 픽 완료 후 수정 및 환불 불가 (블록체인 기록)
✅ 일정 금액 이상 당첨 시 제세공과금 22% 본인 부담
✅ 부정 참여 (다중 계정, 매크로) 적발 시 당첨 취소 및 계정 제한
```

### 2. Select 모드 🎁

**여러 경품 중 원하는 것을 사전에 선택하는 게임**

#### 차이점
- 먼저 **원하는 경품을 선택**하고 참가비 결제
- 모든 참가자가 **동일한 게임 보드** 사용
- 당첨 시 **선택한 경품 수령**
- 다른 사람이 어떤 경품을 선택했는지 알 수 없음

#### 진행 방식
```
1. 경품 리스트 확인
   예: A. 제주 신라호텔 1박 (50,000원)
        B. BTS 콘서트 VIP석 (80,000원)
        C. 도쿄 왕복 항공권 (100,000원)

2. 원하는 경품 선택 및 캐시 차감
   → 이 시점에 경품 확정 (변경 불가)

3. 공유 게임 보드에서 블록 선택
   → 어떤 경품을 선택했는지 다른 사람은 모름

4. Daily 모드와 동일하게 진행

5. 당첨 시 선택한 경품 수령
```

#### 예시
```
Select 게임 #5678
격자: 20×20 (400블록)

참가자 A: 제주 신라호텔 선택 (50,000원) → 블록 123 픽
참가자 B: BTS 콘서트 선택 (80,000원) → 블록 123 픽
참가자 C: 도쿄 항공권 선택 (100,000원) → 블록 45 픽
참가자 D: 제주 신라호텔 선택 (50,000원) → 블록 67 픽

블록 45번: 1명 (참가자 C만 선택) ← 당첨!

결과: 참가자 C가 도쿄 왕복 항공권 획득! 🎉
```

### 3. Vibe 모드 🎨

**경품이 아닌 예술 문화 콘텐츠를 보상으로 지급하는 게임**

#### 특징
- 물리적 경품 대신 **디지털 콘텐츠** 또는 **문화 체험** 제공
- 예술가 지원 및 문화 확산이 목적
- 낮은 참가비로 부담 없이 참여

#### 보상 예시
```
🎨 예술 작품:
- 디지털 아트 NFT
- 작가 친필 사인 포스터
- 한정판 아트북

🎭 문화 체험:
- 갤러리 전시 초대권
- 아티스트 1:1 멘토링
- 작업실 투어

🎵 음악 콘텐츠:
- 미공개 음원
- 아티스트 팬미팅 초대권
- 콘서트 VIP 패키지
```

#### 진행 방식
```
Daily 모드와 동일하나,
보상이 예술/문화 콘텐츠로 제공됨
```

## 사용자 혜택

### 재화 시스템

BlockPick은 **2가지 캐시** 시스템을 운영합니다.

#### 충전 방식
```
실제 돈으로 충전하면 2가지 캐시를 동시에 지급합니다.

예시: 10,000원 충전 시
→ 게임캐시 10,000원 지급
→ 쇼핑캐시 10,000원 지급
(총 20,000원 상당의 재화 획득)
```

#### 게임캐시 (Game Cash)
- **획득**: 실제 돈으로 충전 (충전 금액만큼 지급)
- **용도**: 게임 참여 전용
- **환불**: 게임 참가 이력이 없을 경우에만 가능
- **만료**: 5년

#### 쇼핑캐시 (Shopping Cash)
- **획득**: 실제 돈으로 충전 (충전 금액만큼 지급)
- **용도**: BlockPick 몰에서 여행상품 구매
- **환불**: 게임 참가 이력이 없을 경우에만 가능
- **만료**: 5년

#### 재화 사용 규칙
```
게임 참여:
✅ 게임캐시만 사용 가능
❌ 쇼핑캐시 사용 불가

BlockPick 몰 (여행상품 구매):
✅ 쇼핑캐시 사용 가능
❌ 게임캐시 사용 불가
```

#### 환불 정책
```
⚠️ 중요: 게임 참가 이력이 환불 가능 여부를 결정합니다.

✅ 환불 가능한 경우:
- 충전 후 단 한 번도 게임에 참가하지 않은 경우
- 게임캐시와 쇼핑캐시 모두 환불 가능

❌ 환불 불가능한 경우:
- 한 번이라도 게임에 참가한 이력이 있는 경우
- 게임캐시와 쇼핑캐시 모두 환불 불가
```

### 이벤트 및 프로모션

#### 신규 가입 이벤트
- 가입 축하 캐시 **5,000원** 지급
- 첫 게임 참여 시 **20% 할인**

#### 친구 초대 이벤트
- 친구 가입 시 양쪽 모두 **3,000원** 지급
- 친구가 첫 게임 참여 시 추가 **2,000원**

#### 출석 체크 이벤트
- 7일 연속 출석 시 **1,000원 쇼핑캐시**
- 30일 연속 출석 시 **5,000원 쇼핑캐시**

#### 월간 행운 이벤트
- 매월 최다 참여자 **10,000원 보너스**
- 매월 행운의 1명 **고가 상품 추가 증정**

## 안전 장치

### 부정 방지 시스템

#### IP 및 디바이스 추적
- 동일 IP에서 다계정 생성 제한
- 의심스러운 패턴 자동 감지

#### 실명 인증
- 고가 상품 당첨 시 실명 인증 필수
- 본인 확인 후 상품 발급

#### 관리자 검수
- 모든 당첨 건 관리자 검수
- 의심 사례 발견 시 당첨 취소

### 개인정보 보호

#### 익명성 보장
- 게임 참여 시 닉네임만 노출
- 당첨자 정보 마스킹 처리 (예: 김\*\*)

#### 데이터 암호화
- 비밀번호: bcrypt 해싱
- 중요 정보: AES-256 암호화
- 통신: HTTPS/TLS

#### GDPR 준수
- 개인정보 처리방침 명시
- 데이터 삭제 요청 가능
- 유럽 사용자 별도 보호
