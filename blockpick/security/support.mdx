# 고객 지원 가이드

## 고객 지원 센터

BlockPick은 사용자의 문제를 신속하고 정확하게 해결하기 위해 다양한 지원 채널을 운영합니다.

### 지원 채널

```
📧 이메일: support@blockpick.io
💬 카카오톡: @blockpick
☎️ 전화: 1234-5678
💻 실시간 챗봇: 24시간 운영
🎫 1:1 문의: 마이페이지 > 문의하기
```

## 지원 시간

### 운영 시간

```
평일: 09:00 - 18:00 (주말·공휴일 제외)
점심시간: 12:00 - 13:00

긴급 문의: 24시간 대응
- 결제 오류
- 보안 문제
- 계정 해킹
```

### 응답 시간

#### 일반 문의

```
이메일: 24시간 이내
카카오톡: 2시간 이내 (운영 시간)
전화: 즉시 연결
챗봇: 즉시 (24시간)
1:1 문의: 24시간 이내
```

#### 긴급 문의

```
결제 오류: 2시간 이내
보안 문제: 즉시
계정 잠김: 4시간 이내
```

## 문의 유형별 안내

### 1. 회원 가입 및 계정

#### 자주 묻는 질문

**Q. 회원가입이 안 돼요.**
```
확인 사항:
✅ 이메일 형식이 올바른가요?
✅ 비밀번호가 8자 이상인가요?
✅ 약관에 동의하셨나요?

해결 방법:
1. 다른 이메일로 시도
2. 비밀번호 재설정
3. 브라우저 쿠키 삭제
4. 고객센터 문의
```

**Q. 비밀번호를 잊어버렸어요.**
```
해결 방법:
로그인 페이지 > "비밀번호 찾기"
   ↓
가입 이메일 입력
   ↓
이메일로 재설정 링크 발송
   ↓
새 비밀번호 설정
   ↓
로그인 완료
```

**Q. 이메일 인증이 안 와요.**
```
확인 사항:
✅ 스팸 메일함 확인
✅ 이메일 주소 확인
✅ 10분 대기

해결 방법:
"인증 메일 재발송" 클릭
또는 고객센터 문의
```

**Q. 계정을 탈퇴하고 싶어요.**
```
절차:
마이페이지 > 설정 > 회원 탈퇴
   ↓
본인 인증
   ↓
탈퇴 사유 선택
   ↓
탈퇴 완료 (즉시)

⚠️ 주의:
- 복구 불가능
- 캐시 잔액은 먼저 환불 필요
- 진행 중인 게임 확인
```

### 2. 게임 참여

#### 자주 묻는 질문

**Q. 게임 참여가 안 돼요.**
```
확인 사항:
✅ 캐시 잔액이 충분한가요?
✅ 게임 마감 시간이 지나지 않았나요?
✅ 이미 참여한 게임은 아닌가요?

해결 방법:
1. 캐시 충전
2. 다른 게임 선택
3. 페이지 새로고침
4. 고객센터 문의
```

**Q. 블록을 선택했는데 제출이 안 돼요.**
```
확인 사항:
✅ 인터넷 연결 확인
✅ 블록을 정확히 클릭했나요?
✅ "제출" 버튼을 눌렀나요?

해결 방법:
1. 페이지 새로고침
2. 다시 블록 선택
3. 다른 브라우저 시도
4. 고객센터 문의 (참가비 환불)
```

**Q. 당첨됐는데 알림이 안 와요.**
```
확인 사항:
✅ 마이페이지 > 당첨 내역 확인
✅ 이메일 스팸함 확인
✅ 앱 알림 설정 확인

해결 방법:
마이페이지에서 직접 확인 가능
알림 설정: 마이페이지 > 설정 > 알림
```

**Q. 왜 낙첨됐는지 모르겠어요.**
```
확인 방법:
마이페이지 > 게임 이력 > 상세보기
   ↓
블록별 선택 인원 확인
   ↓
내가 선택한 블록의 인원 확인
   ↓
최소 선택 블록 확인

이해가 안 되면:
"블록체인 검증" 클릭 → 투명하게 확인
```

### 3. 결제 및 환불

#### 자주 묻는 질문

**Q. 결제했는데 캐시가 충전 안 돼요.**
```
확인 사항:
✅ 결제 완료 메시지를 받았나요?
✅ 5분 정도 기다려보셨나요?
✅ 마이페이지 > 결제 내역 확인

해결 방법:
1. 5분 대기 후 재확인
2. 페이지 새로고침
3. 고객센터 문의 (승인 번호 제공)
   → 수동 처리 (10분 이내)
```

**Q. 중복 결제됐어요.**
```
확인:
마이페이지 > 결제 내역에서 확인

해결:
자동 환불: 24시간 이내
수동 환불: 고객센터 문의 (즉시 처리)
```

**Q. 환불은 언제 되나요?**
```
결제 수단별 환불 시간:

신용카드:
- 당일 취소: 즉시
- 일반 환불: 3~7 영업일

계좌이체: 1~3 영업일
간편결제: 1~3 영업일
해외 카드: 5~10 영업일

자세한 내용: 환불 정책 참조
```

**Q. 현금영수증 발급이 안 돼요.**
```
확인 사항:
✅ 계좌이체 또는 간편결제로 충전했나요?
✅ 주민등록번호를 등록했나요?
✅ 충전 후 7일 이내인가요?

해결 방법:
마이페이지 > 결제 내역 > 현금영수증 발급
또는 고객센터 문의
```

### 4. 상품 예약

#### 자주 묻는 질문

**Q. 예약 정보 입력 마감이 지났어요.**
```
상황:
❌ 기한 연장 불가 (원칙)

예외 (불가항력):
✅ 천재지변
✅ 본인 입원 (진단서)

신청:
고객센터로 증빙 자료와 함께 문의
→ 개별 심사
```

**Q. 예약 날짜를 변경하고 싶어요.**
```
상품별 변경 정책:

숙박:
- 체크인 14일 전까지
- 1회 무료, 2회째부터 10,000원

티켓:
- 원칙적으로 불가
- 주최 측 변경 시만 가능

항공권:
- 국내선: 출발 3일 전까지
- 국제선: 출발 7일 전까지

자세한 내용: 예약 정책 참조
```

**Q. 바우처를 못 받았어요.**
```
확인:
✅ 이메일 스팸함 확인
✅ 마이페이지 > 당첨 내역에서 다운로드

해결:
고객센터 문의 → 즉시 재발송
```

**Q. 예약이 거부됐어요.**
```
거부 사유:
- 실명 불일치
- 연락처 오류
- 여권 정보 오류 (항공권)
- 예약 불가 날짜

해결:
재입력 기회 (최대 3회)
고객센터 문의 → 정확한 사유 안내
```

## 셀프 해결 가이드

### 자주 발생하는 문제

#### 로그인 오류

```
증상: "로그인 실패" 메시지

원인 및 해결:
1. 비밀번호 오류
   → 비밀번호 찾기

2. 계정 잠김 (5회 실패)
   → 30분 대기 또는 고객센터 문의

3. 이메일 오류
   → 가입 시 사용한 이메일 확인
```

#### 페이지 오류

```
증상: 페이지가 로드되지 않음

해결 방법:
1. 페이지 새로고침 (F5)
2. 캐시 삭제
   - Chrome: Ctrl+Shift+Del
   - Safari: Cmd+Option+E
3. 다른 브라우저 시도
4. 인터넷 연결 확인
```

#### 결제 오류

```
증상: "결제 실패" 메시지

원인 및 해결:
1. 카드 한도 초과
   → 카드사 앱에서 한도 확인

2. 잔액 부족
   → 계좌 잔액 확인

3. 카드 정보 오류
   → 카드 번호, 유효기간 재확인

4. 3D Secure 실패
   → SMS 인증 재시도
```

## 자주 묻는 질문 (종합)

### 계정 및 인증

**Q. 본인 인증은 필수인가요?**
A. 일반 게임은 선택이지만, 고가 상품 당첨 시 필수입니다.

**Q. 미성년자도 이용 가능한가요?**
A. 만 14세 이상 가능합니다. 만 14세 미만은 법정대리인 동의가 필요합니다.

### 게임 관련

**Q. 당첨 확률은 얼마나 되나요?**
A. 게임 크기와 참여자 수에 따라 다릅니다. 평균 3~7% 정도입니다.

**Q. 게임 결과를 조작할 수 있나요?**
A. 불가능합니다. 블록체인과 스마트 컨트랙트로 완전히 투명하게 운영됩니다.

### 결제 및 환불

**Q. 최소 충전 금액은 얼마인가요?**
A. 국내 1,000원, 해외 $10입니다.

**Q. 해외 결제도 가능한가요?**
A. 네, Stripe을 통해 135개 이상 통화로 결제 가능합니다.

### 상품 관련

**Q. 당첨된 상품을 현금으로 바꿀 수 있나요?**
A. 아니요, 상품은 현금 전환이 불가능합니다.

**Q. 예약 기한을 연장할 수 있나요?**
A. 원칙적으로 불가능하지만, 불가항력 사유(입원, 천재지변 등)는 개별 심사합니다.
