# BlockPick - 블록체인 기반 투명한 행운 플랫폼

## 환영합니다! 🎉

BlockPick은 블록체인 기술을 활용한 완전히 투명하고 공정한 행운 게임 플랫폼입니다. 게임에 당첨되면 **숙박권, 공연/스포츠 티켓, 항공권** 등 다양한 상품을 받을 수 있습니다.

## 왜 BlockPick인가요?

### 🔐 완벽한 투명성
모든 게임 결과가 Polygon 블록체인에 기록되어 조작이 불가능합니다. 누구나 PolygonScan에서 직접 검증할 수 있습니다.

### ⚖️ 공정한 게임
Commit-Reveal 암호학 기법으로 다른 사람의 선택을 보고 따라하거나, 선택을 나중에 바꾸는 것이 원천적으로 차단됩니다.

### 🎁 다양한 경품
- **숙박권**: 국내외 프리미엄 호텔, 리조트, 펜션
- **티켓**: 콘서트, 뮤지컬, 스포츠 경기, 전시회
- **항공권**: 국내선, 국제선 (일반석/비즈니스석)
- **예술/문화**: NFT, 전시 초대권, 미공개 음원 (Vibe 모드)

### 💰 낙첨자도 혜택
낙첨하더라도 픽당 가격의 **10%를 포인트로 환급**해드립니다. 포인트는 BlockPick 몰에서 현금성 할인으로 사용할 수 있습니다. (게임 재참여는 불가)

## 빠른 시작

### 1. 회원가입 (30초)
이메일과 닉네임만으로 간편하게 가입하세요.

### 2. 재화 충전
토스페이먼츠를 통해 안전하게 충전하세요. (최소 1,000원)
※ 충전 시 게임캐시와 쇼핑캐시를 각각 충전 금액만큼 지급! (2배 혜택)

### 3. 게임 참여
원하는 게임을 선택하고 격자에서 블록을 클릭하세요!

### 4. 결과 확인
커밋-리빌 단계가 끝나면 자동으로 당첨자가 결정됩니다.

### 5. 상품 예약
당첨되면 원하는 날짜에 예약하여 사용하세요!

## 주요 기능

### 🎮 블록픽 게임
- 3x3 ~ 10x10 다양한 격자 크기
- 참가비 1,000원 ~ 100,000원
- 가장 적은 인원이 선택한 블록의 참여자가 당첨!

### 💳 편리한 결제
- **국내 결제**: 토스페이먼츠 (카드, 계좌이체, 간편결제)
- **해외 결제**: Stripe (신용카드, PayPal, Apple Pay)
- 캐시 + 포인트 혼합 결제 가능

### 📱 실시간 알림
- 게임 시작/종료 알림
- 당첨 결과 알림
- 예약 확정 알림
- 마감 임박 알림

## 서비스 흐름

```
회원가입 → 캐시 충전 → 게임 참여 → 당첨! → 예약 → 상품 이용
                    ↓
                  낙첨 → 포인트 적립 → 다시 도전!
```
